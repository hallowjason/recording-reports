# SVG 風格簡報 — 樞紐慈悲報告

所有 SVG 必須嚴格遵守以下設計語言。主題：道場全球弘法戰略，調性莊嚴、溫暖、有靈性，**絕不卡通、絕不廉價、絕不用 emoji**。

## 視覺核心：暖米色底 + 金線纏繞
- 整體像一張溫潤的米色宣紙／羊皮紙，上面以**極細的金線**勾勒、纏繞、編織出圖形。
- 金線是主角：線條要有「絲線纏繞、流動、交織」的手感，可用多條平行細線、貝茲曲線交纏、節點匯聚等手法。
- 質感參考：金箔、燙金、絲線刺繡、星圖連線。寧靜、精緻、留白多。

## 色票（只准用這些顏色）
- 底色 紙米色 `#F5EEE0`（SVG 背景請用透明 `transparent`，讓網頁底色透出，除非另有指示）
- 深米卡片 `#EFE6D4`
- 墨棕（主文字/深線） `#2A2017`
- 柔棕（次要） `#6B5D4A`
- 金 主色 `#B8893B`
- 亮金 `#D9B978`
- 深金 `#8C6A2A`
- 細線金 `#C9A85F`
- 陶土（次要強調，僅少量點綴） `#C2703D`
- 髮絲線 `#D8C9AC`

## 線條與幾何
- 細線：stroke-width 0.75–2，金線為主。重點輪廓最多 2.5。
- 幾何語彙：圓形與膠囊（pill）為主，呼應「圓融」。節點用小圓點（金色）。
- 多用 `stroke` 描邊勝於大面積 `fill`；填色僅用於極小的節點或極淡的暈染。
- 可用極淡的徑向暈染（金色 opacity 0.06–0.12）作為光暈。

## 技術規範
- 輸出**單一個 `.svg` 檔**，純 SVG（含 `<svg xmlns>`），不要 HTML 包裝、不要 `<script>`。
- 一律設定 `viewBox`，不要寫死 width/height（除非指定）。
- 背景透明（不畫滿版底色矩形），除非該圖指定要底色。
- 所有顏色用上面色票的 hex，不要引入其他色相（不要藍、綠、紫、粉）。
- 線條可用 `stroke-linecap="round"`、`stroke-linejoin="round"` 求柔和。
- 不要嵌入點陣圖、不要外部字型；若需文字，用 system serif 並轉為次要。
- 圖面要**精緻克制、留白充足**，不要塞滿。
